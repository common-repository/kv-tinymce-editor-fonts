A tiny Plugin to add a new font to your post editor i.e TinyMCE Editor. 

== Description ==

 A Simple and Small plugin helps to select your desired fonts from Google Webfonts and which can be used in your website. There are more amount of fonts available. So most of the google webfonts available to select. You can select your desired font and which will be available to work in editor and also its available in your site front end. Hope you will enjoy it.
 If you want to rework with the code, fine take a look at here before start usging this code, It will be helpful to work with your own project.<a href="http://kvcodes.com/2014/05/how-to-add-google-webfonts-to-wordpress-tinymce-editor/" target="blank" > Kvcodes.com </a> 


= Features =
* Easy use
* Easy to change your fonts 
* Simple Set-up, Less space consuming
* Google Web Fonts added
* You can have an option to enable and disable default fonts list

 

== Installation ==

1. Upload `kv-tinymce-editor-fonts` to the `/wp-content/plugins/`  directory
2. Activate the plugin through the 'Plugins' menu in WordPress
